# VoiceEmotion

사용자의 기분에 따라 알맞는 영화 추천 시스템

---

MFCC를 이용하여 사용자의 기분을 파악한 후 현재 기분을 풀기 위해 알맞는 미디어 추천

## Environment

- Linux ( Docker )

### Django

- Python 3.8

### Modeling

- Python 3.6
- tensorflow
- librosa
